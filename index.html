Contextlogger

Context Logger App for Android devices by CARES in SNU

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Github Pages for Context Logger.

This projects is to develop an android application for collecting context logs of a mobile device. This project has been initiated from CARES (Computer ARchitecture and Embedded Systems Lab.) in Seoul National University.

Download and test like follows! :

$ cd YOUR_WORK_DIR
$ git clone https://github.com/myshepherd/contextlogger.git

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch. You can also be a contributor. Please contact me if you are interest with.

Project Details

  1. Logger App : Android App for logging context

  2. Logger Repository : Repository, a host side application, for collecting logged data

It provides a server program running on a host side (like linux-or-windows-based pc).

  1. Logger Analyzer : Analyzer, a host side application, for logged context data It analysis a db file collected from a logger repository.

How to build?

  1. Logger App : ContextLogger Directory

It can be compiled using Android Studio or Eclipse like general other android applications.

  1. Logger Repository & Logger Analyzer : ContextLog Directory

They provide their own Makefile. So just,

$ make

How to run?

  1. Logger App

Click "On" Button in the main activity of the app.

  1. Logger Repository & Logger Analyzer : ContextLog Directory

They provide its own shell-script file. Detail options like follows:

usage: LoggerRepository [Options]
 -p <port>   [p]ort number

usage: ContextLogAnalyzer filename [Options]
 -i                      [i]nclude logger on/ogg message
 -o <Output directory>   Indicate [o]utput directory
 -T                      Make output using human-readable [T]ime format
